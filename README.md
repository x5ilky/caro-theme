# Caro Theme
*Caro* is an ORAC theme for Tampermonkey.

## Installation
Install _Tampermonkey_ ([Chromium](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en), [Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)) from the web stores. Create a new userscript and paste in the example in [`tampermonkey.example.js`](./tampermonkey.example.js). Press `Ctrl+S` or `Cmd+S` to save and enjoy!
